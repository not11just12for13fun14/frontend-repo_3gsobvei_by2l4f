# ECLIPTZ TECHSTORM HACKATHON 2025 — Website

A premium, neon-futuristic, fully responsive site for the 24-hour hackathon by Ecliptz Labs.

## Run locally
- npm install
- npm run dev
- Open the provided URL

## Test the core flow
1. Go to Registration → fill the form → Submit
2. You are redirected to Payment. Click "Pay Now (Dummy)"
3. Click "Payment Success → Continue" to see Submission Confirmation with your data

All state is stored in localStorage/sessionStorage. No backend required.
